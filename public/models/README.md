# GLB Modelleri Klasörü

Bu klasör, oyununuzda kullanmak istediğiniz özel 3D modelleri (.glb formatında) manuel olarak yüklemeniz için oluşturulmuştur.

## Nasıl Kullanılır?

1. Kendi `.glb` dosyalarınızı bu klasörün (`public/models/`) içine sürükleyip bırakın.
2. Oyunun Admin Paneline giriş yapın.
3. **GLB Varlıklar** sekmesinde, değiştirmek istediğiniz modelin karşısındaki kutucuğa dosyanın yolunu yazın.
   - Örnek: Eğer dosyanızın adı `benim-arabam.glb` ise, kutucuğa `/models/benim-arabam.glb` yazın.
4. "AYARLARI KAYDET" butonuna tıklayın.

Oyun otomatik olarak bu klasördeki dosyayı okuyacak ve oyunda gösterecektir.
